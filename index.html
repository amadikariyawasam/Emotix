Project Abstract

What is EMOTIX?

EMOTIX is an AI-based workplace system designed to enhance employee mental wellness and productivity while protecting privacy. It monitors individual work patterns, detects attention and fatigue levels through eye tracking, and recognizes stress by analyzing facial expressions over time. The system processes data locally on edge devices to ensure privacy, encrypts sensitive emotional information, and includes chatbot support for employee assistance. It provides a secure, intelligent platform for proactive workplace wellness management in modern smart office environments.

Core Research Components

👁️

AI-based eye focus detection & adaptive learning assistant

L.W.D.K Perera - IT22601810

OVERVIEW

A non-intrusive, wearable-free focus monitoring system that captures real attention rather than just presence. Combines eye tracking with environmental sensing for context-aware focus scoring — all in real-time.

HOW IT WORKS

  1. Facial landmark detection: OpenCV + MediaPipe FaceMesh identifies 468 facial landmarks in real-time video streams per frame.
  2. Eye metrics extraction: Eye Aspect Ratio (EAR) detects blink patterns and fatigue. Gaze direction and fixation duration assess sustained attention and distraction events.
  3. Environmental context integration: Light, noise, and temperature sensor data are fused with eye metrics. High noise correlates negatively with focus duration; poor lighting increases blink rate.
  4. Focus score computation: focus = (w1*f) + (w2*e) + (w3*g) + (w4*d), normalized 0-1. Weighted penalties applied for non-center gaze, high blink frequency, and short fixations.
  5. Dashboard & adaptive feedback: Real-time focus score drives an intelligent dashboard. AI assistant generates personalized concentration reminders and wellness break requests.

VALIDATION METRICS

Focus classifier accuracy, precision, recall, F1-score. Time-series analysis detects fatigue patterns and peak distraction periods across long work sessions.

OpenCV MediaPipe FaceMesh EAR Algorithm Gaze estimation Environmental sensing Weighted scoring Personalized Feedback

Real-time secure AI stress detection

K.G.A.N. Kariyawasam - IT22927170

OVERVIEW

A non-invasive, privacy-preserving stress monitoring pipeline that runs entirely on edge hardware. Replaces EEG/ECG wearables with facial video analysis, ensuring no raw emotional data leaves the device.

HOW IT WORKS

  1. Preprocessing: Facial images converted to 48x48 grayscale normalized inputs from live camera feed.
  2. CNN emotion recognition: CNN trained on FER+ dataset classifies 8 emotion categories per frame, outputting probability vectors.
  3. LSTM temporal analysis: A 15-frame sliding buffer feeds an LSTM model that captures emotional transitions over time, predicting valence and arousal values.
  4. Stress score generation: Valence-arousal values are mapped to the Russell Circumplex Model to compute a normalized stress score, stronger than single-frame analysis.

SECURITY LAYER

AES-256-GCM encryption protects all emotional data in transit. HMAC-SHA256 ensures integrity and tamper detection. IV reuse is prevented using device MAC address + message counters.

VALIDATION METRICS

Classification accuracy, confusion matrix, precision, recall, and temporal consistency. Stress score trends analyzed per session to identify high stress periods. The system filters momentary facial micro-expressions to maintain long-term predictive stability. Performance is validated across diverse environmental lighting and facial feature variations. The model is specifically optimized for ARM-based edge processors to ensure high-frequency real-time inference.

CNN LSTM FER+ dataset Russell Circumplex AES-256-GCM HMAC-SHA256 Edge computing

Emotix honeypot — deception-based attack detection

S.A.V.P. Jayarathna - IT22560612

OVERVIEW

A lightweight, deception-based cybersecurity layer designed to protect the EMOTIX IoT and embedded infrastructure. Lures attackers into fake services, captures malicious behavior, and switches between passive and active modes.

HOW IT WORKS

  1. Decoy services deployment: Fake login interfaces and decoy HTTP & MQTT services are exposed to attract attackers away from real system seeds.
  2. Passive traffic monitoring: MQTT and HTTP traffic is continuously observed. Frequency distributions and anomaly scores are computed per request to build a behavioral baseline.
  3. Isolation Forest anomaly detection: ML model detects anomalous behavior including SQL injection, XSS, and path traversal attacks. Validated using True Positive Rate (TPR), False Positive Rate (FPR).
  4. Active deception mode: When a threat is confirmed, the system switches from passive to active mode — engaging the attacker in fake interactions to capture intent.

INTEGRATION WITH EMOTIX SECURITY

Works alongside AES-256-GCM encryption and HMAC-SHA256 integrity checks to form a complete security layer. Protects sensitive employee wellness data and stress scores.

VALIDATION METRICS

True Positive Rate (TPR), False Positive Rate (FPR), detection latency, deception mode activation frequency, and comparative rule-based vs AI-based detection performance.

Adaptive routine monitoring & wellness support

R.M.H.D. Rathnayaka - IT22560476

OVERVIEW

A TinyML-powered system that learns each employee's personal work routine on-device and detects significant behavioral deviations — without cloud dependency or preset schedules. All routine changes require employee consent before being permanently applied.

HOW IT WORKS

  1. Observation phase (1-2 weeks): Raspberry Pi continuously logs work start time, break times, lunch time, and end of day patterns per employee.
  2. Routine modeling: TinyML model builds a personal routine baseline using descriptive statistics — mean (μ) and standard deviation (σ) for each daily event.
  3. Deviation detection: Inference model μ ± max(1.5σ, 45min) differentiates natural flexibility from significant routine changes, triggering wellness cues when crossed.
  4. AI chatbot interventions: Rule-based chatbot converts routine and focus analytics into plain language recommendations: break reminders, refocus nudges, workplace adjustments.

HUMAN-IN-THE-LOOP CONSENT

Unlike systems that auto-enforce schedules, EMOTIX requires employee approval before updating a routine permanently. This builds trust, autonomy, and ethical acceptance in the workplace.

VALIDATION METRICS

Presence data, accuracy, deviation detector reliability, chatbot intervention acceptance rate, and behavioral change post-intervention via user surveys.
