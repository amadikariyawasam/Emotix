Project Abstract

What is EMOTIX?

EMOTIX is an AI-based workplace system designed to enhance employee mental wellness and productivity while protecting privacy. It monitors individual work patterns, detects attention and fatigue levels through eye tracking, and recognizes stress by analyzing facial expressions over time. The system processes data locally on edge devices to ensure privacy, encrypts sensitive emotional information, and includes chatbot support for employee assistance. It provides a secure, intelligent platform for proactive workplace wellness management in modern smart office environments.

Key Features

Core Research Components

Four integrated modules form the EMOTIX ecosystem, each tackling a unique dimension of workplace mental wellness.

🧠

Real-Time Secure AI Stress Detection

Analyzes facial expressions using CNN + LSTM deep learning models on a privacy-preserving edge device. Detects temporal stress patterns mapped through the Russell Circumplex valence-arousal model — all without uploading to the cloud.

🔄

Adaptive Routine Monitoring & Wellness Support

Uses TinyML on Raspberry Pi to learn individual employee work patterns over time. An AI chatbot converts behavioral analytics into personalized wellness recommendations and proactive interventions.
