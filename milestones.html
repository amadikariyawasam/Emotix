Research Timeline

All major project milestones from initial proposal through to the final viva examination.

📋 Project Proposal

The project proposal marks the formal initiation of the EMOTIX research project. This milestone involves presenting the research idea, problem statement, objectives, proposed methodology, and a preliminary literature review. The team must demonstrate feasibility, novelty, and academic merit of the proposed AI-based wellness system.

Deliverables: Proposal Document Deliverables: Presentation Slides Deliverables: Project Charter

📊 Progress Presentation I

Date: [ Add Date ] Marks: [ Add ] %

The first progress presentation evaluates the initial implementation phase of the EMOTIX system. Teams are expected to demonstrate working prototypes of at least one or two core components — such as the edge-based data collection module or the preliminary stress detection pipeline. Progress against objectives is assessed against the timeline.

Deliverables: Progress Report Deliverables: Working Prototype Demo Deliverables: Checklist

📈 Progress Presentation II

Date: [ Add Date ] Marks: [ Add ] %

The second progress presentation showcases the near-complete integrated system. All four EMOTIX components — stress detection, eye tracking, routine monitoring, and chatbot support — should be integrated and tested at this stage. Evaluation includes system performance, accuracy metrics, and security features.

Deliverables: Updated Progress Report Deliverables: Integration Demo Deliverables: Test Results

🏁 Final Assessment

Date: [ Add Date ] Marks: [ Add ] %

The final assessment is the comprehensive evaluation of the completed EMOTIX system. This includes a full system demonstration, evaluation of all research objectives, analysis of results against hypotheses, and submission of the final research report. The system must meet all privacy, security, and performance benchmarks.

Deliverables: Final Report Deliverables: Full System Demo Deliverables: Research Paper Draft

🎓 Viva Voce

Date: [ Add Date ] Marks: [ Add ] %

The Viva Voce is the oral examination where each team member must individually defend their research contributions, design decisions, and implementation details. Examiners will probe technical depth, individual understanding, and the team's ability to critically evaluate EMOTIX against existing literature and industry standards.
